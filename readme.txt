== Frequently Asked Questions ==

= Can I use Super Addons For Elementor without Elementor? =

You cannot use **[Super Addons For Elementor](https://www.superaddons.themehat.com/)** without Elementor. Super Addons For Elementor is a collection of slick, powerful widgets that works seamlessly with Elementor page builder and it's only for Elementor.

= Does it work along with other Elementor Addons? =

Yes, it does.

= Does it work with any WordPress theme? =

Yes, it works with any WordPress theme that works with Elementor. And it best works with [Hello Elementor](https://wordpress.org/themes/hello-elementor/).

= Does it work with Elementor Pro? =

Yes, undoubtedly.

= Will Super Addons For Elementor break my site after an update? =

No, It won't break your site or any page where you used Super Addons For Elementor. We put our best effort to make you happy.


== Installation ==

**Step 1:** Upload the plugin file to install by navigating through Plugins➔Add New➔Upload Plugin➔Choose File to Install from your WordPress dashboard or you can simply search for the Super Addons For Elementor from the plugin directory by going to Plugins➔Add New and search for the plugin in the search tab to install it on your WordPress site.

**Step 2:** After successful installation, you have to click the "activate" button to activate the Super Addons For Elementor.

**Step 3:** When you activate Super Addons For Elementor you will Get The Widget In The Elementor Editor.
